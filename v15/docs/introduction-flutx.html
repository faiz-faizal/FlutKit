Introduction

FlutX is a finely designed and developed package using Flutter. This package is exclusively provided by Coderthemes.

There are many plugins available on https://pub.dev/ and many other sources. But our package will allow you to add any widget to your application in just one word and further add customisation to it as per your convinience. Moreover, it is facilitated with TWO TONE ICONS which are provided exclusively by our package, which allows you to use all the Icons in two tone.

It also provides different theme, utility, style and route widgets.
